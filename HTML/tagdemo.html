html structure demo

<html> tag

the <html> tag is the root of an html document.It wraps the entire web page code.

<head> tag

the<head> tag contains meta-information about the document,such as the title, character set,links to stylesheets,etc.

<title> tag

the<title>tag defines the title of the page. You can see it in the browser tab. for the page, it is set as: HTML Tags Demo- Chipi Technologies.

<body> tag

the <body> tag contains all the visible content of the web page like headings,text,images,links,lists,etc.Everything you see here is inside the <body> tag.
